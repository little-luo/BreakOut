# BreakOut 打磚塊遊戲

## 🎮 專案簡介
使用 HTML5 Canvas + 原生 JavaScript 開發的小型打磚塊遊戲，實作遊戲物理碰撞與計分邏輯。

## 🔧 使用技術
- HTML5 Canvas API
- Vanilla JavaScript (ES6 Class)
- Object-Oriented Programming
- DOM 操作
- Collision Detection
